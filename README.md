# ⚡ Tesla Inventory System

A modern React-based web app to browse Tesla vehicles with authentication, theme switching, and smooth UI animations.

---

## 🚀 Live Demo
View the live version here:  
👉 https://teslaclone-pink-theta.vercel.app/

---

## ✨ Features

- 🚗 Browse Tesla vehicle listings with details  
- 🔐 User login & registration with saved sessions  
- 🌓 Dark / light mode toggle  
- 📱 Fully responsive design  
- 🎨 Smooth animations and transitions  
- ⚡ Fast navigation with React Router  

---

## 🛠️ Tech Stack

- React.js  
- React Router  
- Bootstrap / React Bootstrap  
- AOS (Animations)  
- localStorage
